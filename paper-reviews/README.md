# Paper Reviews

연구 관심사와 직접 연결되는 논문을 읽고 정리하는 공간입니다.

## Review Focus

| Topic | Why it matters |
| --- | --- |
| Autonomous driving perception | 복잡한 도로 환경에서 인식 결과를 어떻게 신뢰하고 활용할지 이해하기 위해 |
| Road topology & ego-lane inference | 차선 혼잡 구간에서 ego-lane 판단 문제를 연구 질문으로 발전시키기 위해 |
| Trajectory prediction | 주변 차량의 미래 움직임 예측 문제와 sequence model 활용을 이해하기 위해 |
| NLP / LLM for policy analysis | DRT 정책 제안 프로젝트의 방법론과 검증 기준을 세우기 위해 |

## Review Template

```text
Date:
Title:
Authors:
Venue / Year:
Link:
Topic:

## Problem
## Method
## Main Contribution
## Limitation
## Connection To My Research
## Questions
## Next Action
```

## Rule

- 논문 내용을 그대로 번역하지 않습니다.
- 내 연구 질문과 연결되는 부분을 중심으로 정리합니다.
- 이해하지 못한 부분을 질문으로 남깁니다.
