# DRT Policy Recommendation

DRT 운행 정책 제안 프로젝트와 논문화 진행 과정을 정리하는 공간입니다.

## Project Summary

기본적인 NLP부터 LLM까지 다룬 자연어 처리 학습을 바탕으로, DRT 운행 정책을 분석하고 정책 제안으로 연결하는 프로젝트입니다.

## Goal

- DRT 운행 정책과 관련된 자료를 정리합니다.
- NLP/LLM을 활용해 정책 제안에 필요한 근거를 구성합니다.
- 프로젝트 진행 과정과 논문화 과정을 기록합니다.
- 참고 자료와 직접 작성한 분석 내용을 구분합니다.
