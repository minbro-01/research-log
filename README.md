# Research Log

공부한 내용, 프로젝트 진행 과정, 시행착오를 시간순으로 정리하는 저장소입니다.

완성된 결과물만 남기기보다, 어떤 질문에서 시작했고 무엇을 시도했으며 어떤 점을 배웠는지를 기록하는 것이 목적입니다.

## What Goes Here

| Type | Description |
| --- | --- |
| Study Notes | 강의, 책, 문서, 논문을 공부하며 남긴 요약 |
| Project Logs | 프로젝트 진행 과정과 의사결정 기록 |
| Experiment Notes | 실험 조건, 결과, 실패 원인, 다음 시도 |
| Paper Progress | 논문화 과정에서의 자료 정리, 실험, 수정 기록 |
| Retrospectives | 일정 기간 동안 배운 점과 개선할 점 |

## Source Policy

- 수업자료나 외부 자료를 그대로 복사하지 않습니다.
- 참고한 자료는 출처와 함께 남깁니다.
- 내가 이해한 내용, 구현한 내용, 프로젝트에서 맡은 내용을 구분해서 기록합니다.
