# Literature Notes

논문 외의 자료를 읽고 연구와 프로젝트에 필요한 내용을 정리하는 공간입니다.

## Sources

| Source Type | Example |
| --- | --- |
| Technical docs | framework, library, model documentation |
| Lecture materials | 수업 내용 중 연구 질문과 연결되는 부분 |
| Policy documents | DRT, transportation, mobility 관련 정책 자료 |
| Reports | 교통/자율주행/모빌리티 관련 보고서 |

## Note Template

```text
Date:
Source:
Link:
Topic:

## Summary
## Useful Points
## Limitations
## How It Connects
## Next Action
```

## Rule

- 자료의 출처를 남깁니다.
- 단순 요약보다 내 프로젝트/연구에 어떻게 연결되는지를 기록합니다.
