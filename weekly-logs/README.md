# Weekly Logs

한 주 동안 읽은 자료, 진행한 작업, 다음 행동을 짧게 정리하는 공간입니다.

## Weekly Log Format

```text
# YYYY-MM-DD Weekly Log

## Read

## Built / Organized

## Learned

## Questions

## Next Week
```

## Rule

- 완성된 결과만 쓰지 않습니다.
- 어떤 질문이 생겼고 다음에 무엇을 할지 남깁니다.
- 프로젝트 결과물은 `academic-projects`, 논문화 진행은 `paper-progress`에 연결합니다.
